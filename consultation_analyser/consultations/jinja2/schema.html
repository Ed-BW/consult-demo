Consultation data schema v0.1

This page lists the data components required for a consultation to be suitable for processing by the analyser.

Data should be provided in one JSON file containing both the consultation itself and a list of responses. That overall data file is described here and an example is given. Underneath are details of each section, including individual JSON schemas for both the Consultation and ConsultationResponseList parts of the JSON file.
